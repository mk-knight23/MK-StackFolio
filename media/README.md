# Media drop-in (this site)

Save files here with these EXACT names — they appear automatically.
Until a file exists, an animated gradient fallback shows, so nothing looks broken.

## Videos (muted, looping, H.264 .mp4, <6MB each)
- hero-bg.mp4      hero full-screen background (1920x1080, 8-15s)
- footer-bg.mp4    footer background (1920x1080, 8-15s)
- section-1.mp4    optional extra section bg (tag a section with data-cine-bg="section-1.mp4")
- section-2.mp4    optional extra section bg

## Images
- hero-poster.jpg  still shown before hero video loads (1920x1080)
- image-1.jpg .. image-6.jpg  reserved gallery/showcase stills (mapped in when your folder arrives)
