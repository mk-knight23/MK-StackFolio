# Fullstack Product Builder Portfolio — Kazi Musharraf

Personal portfolio with a cinematic, motion-driven UI (full-screen video
backgrounds, glassmorphism, liquid mouse-trail effect).

## Develop
```bash
npm install
npm run dev      # local dev
npm run build    # production build -> dist/
```

## Media
Drop your background videos / images into `public/media/` using the exact
names listed in `public/media/README.md`. They appear automatically; until
then an animated gradient fallback is shown.

Deployed via Vercel (see vercel.json).
